# KOSIS 홀드아웃 100건 평가 - holdout100_v2_development

## 결론

- 기존 개발용 골드 100건과 claim_id 및 article_id 중복 0건
- 평가 코드북: `kosis_codebook_v2.py`
- 물가·고용·무역·인구·소매 각 20건
- 골드 검증 가능: 33건 / 검증 불가: 67건 ({'KOSIS 미제공': 35, '정보 부족': 24, '지역·분류 불일치': 8})

- 80% 품질 기준: 개발셋 통과(독립 재평가 필요)
- 이 표본의 오류를 보고 코드북을 수정했으므로 더 이상 독립 평가셋이 아니다.
